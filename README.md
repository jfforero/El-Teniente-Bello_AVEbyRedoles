El Teniente Bello
Overview

This project presents a web-based audiovisual experience built from the integration of a poetic text and an affective computing system. It is based on El Teniente Bello, a work by Chilean singer-songwriter Mauricio Redolés Bustos, which recounts —through both lyrics and poetic declamation— the disappearance of the pilot Alejandro Bello Silva in 1914, during a flight test to qualify as an aviator.

The declaimed voice of the poem serves as input to an immersive environment generation system (AVE), enabling the transformation of spoken language into dynamic audiovisual compositions.

Project Link

System

The audiovisual material in this repository was generated using the AVE system:

👉 https://huggingface.co/spaces/jfforero/Bello

AVE (Affective Virtual Environments) is a system that generates 3D immersive environments based on speech emotion recognition. It analyzes vocal features and maps affective states into visual and spatial transformations, producing an evolving audiovisual landscape.

Concept

This project explores the relationship between:

poetic language and historical memory
voice and affective expression
speech and immersive audiovisual environments

By using the declamation of El Teniente Bello as input, the system reinterprets the narrative through emotional modulation, creating a layered experience where meaning emerges from both semantics and prosody.

Output

The result is a collection of audiovisual fragments and interactive environments derived from segments of the spoken poem. These outputs can be experienced as:

generative visual sequences
immersive 360° environments
audiovisual installations (e.g., projection mapping)
